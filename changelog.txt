# Zombie Survivor mc1.19.3-forge Changelog

0.1
====
 - 0.1: 하루가 지날 때마다 더 많은 좀비가 나타납니다.
        -- 서바이벌 상태의 각 플레이어 주위에 기준 좌표를 세웁니다.
        -- 각 기준 좌표의 y 좌표가 주변 좌표에 비해 낮다면 해당 기준 좌표는 좀비를 스폰하지 않습니다.
        -- 통과된 각 기준 좌표 별로 근처 랜점 좌표를 여러개 설정합니다,
        -- 랜덤 좌표 별로 좀비 소환 조건이 맞으면 (바이옴, 밝기, 플레이어와의 거리 등..) 좀비 스폰을 시도합니다.
        -- 기준 좌표 별 랜덤 좌표 갯수는 날마다 30씩 증가합니다.

0.2
====
 - 0.2: 좀비 스폰 밸런스 관련 콘피그 파일을 추가했습니다.
        -- Spawn Count Type: 좌표 별 좀비 스폰 시도 횟수 설정 방법 (DEFAULT, RANDOM) 기본값: DEFAULT
        -- Spawn Increase Amount: 날이 지날 때, 증가되는 좀비 스폰 량 (0 ~ 10000) 기본값: 10
        -- Random Coord Type: 각 기준 좌표에서 랜덤 좌표를 설정 방법 (TRIANGLE, UNIFORM) 기본값: TRIANGLE
        이제 가끔씩 주민 좀비가 스폰됩니다.
 - 0.2.1: 모드 업데이트 체크 기능 추가
 - 0.2.2: 플레이어 조건을 관전자 모드만 제외로 변경
          좀비 스폰량 관련 버그 수정 및 Spawn Increase Amount 기본값을 50으로 변경
          이제 다른 몬스터도 스폰됨. 콘피그 파일에서 스폰 비율 설정 가능

0.3
====
 - 0.3.0: 좀비 인식 사거리 강화
 - 0.3.1: 몬스터 스폰 알고리즘 약간 수정
