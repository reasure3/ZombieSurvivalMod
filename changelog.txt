# Zombie Survivor mc1.19.3-forge Changelog

0.x
====
 - 0.1: 하루가 지날 때마다 더 많은 좀비가 나타납니다.
        -- 서바이벌 상태의 각 플레이어 주위에 기준 좌표를 세웁니다.
        -- 각 기준 좌표의 y 좌표가 주변 좌표에 비해 높다면 해당 기준 좌표는 좀비를 스폰하지 않습니다.
        -- 통과된 각 기준 좌표 별로 근처 랜점 좌표를 여러개 설정합니다,
        -- 랜덤 좌표 별로 좀비 소환 조건이 맞으면 (바이옴, 밝기, 플레이어와의 거리 등..) 좀비 스폰을 시도합니다.
        -- 기준 좌표 별 랜덤 좌표 갯수는 날마다 30씩 증가합니다.
